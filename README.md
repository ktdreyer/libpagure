# libpagure

A Python library for Pagure APIs. Pagure is a light-weight git-centered forge based on pygit2, created by Pierre-Yves Chibon.

## Installation

Use pip to install.

### Linux

```
pip3 install libpagure
```

### OS X

```
python3 -m pip install libpagure
```
